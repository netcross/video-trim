# server.py find_ffmpeg()

### 2. 보수적 분할 전략 (High #1 대응) — ✅ 구현 완료

```

### 3. 저장 경로 처리 (Medium #4 대응) — ✅ 구현 완료 (v1.1)
```
v1.0: 텍스트 입력으로 경로 직접 지정 + 서버 사이드 유효성 검증
